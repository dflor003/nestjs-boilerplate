# NestJS Boilerplate

Boilerplate repo for a [NestJS](https://github.com/nestjs/nest) service.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# Development
$ npm run start
$ npm run start:dev

# Production mode
$ npm run start:prod
```

## Test

```bash
# Unit tests
$ npm run test

# E2E tests
$ npm run test:e2e

# Test coverage
$ npm run test:cov
```
